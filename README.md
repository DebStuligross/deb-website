# Deb Stuligross — Personal Professional Site

## Files
- `index.html` — the page
- `headshot.png` — photo

## Publishing to GitHub Pages

1. Create a new GitHub repo (e.g. `deb-stuligross` or `debstuligross.github.io`)
2. Upload both `index.html` and `headshot.png` to the repo root
3. Go to **Settings → Pages**
4. Set Source to: **Deploy from a branch**, branch **main**, folder **/ (root)**
5. Save — the site will be live at `https://[your-username].github.io/[repo-name]` within a minute or two

## Updating the page
Edit `index.html` directly in GitHub (pencil icon) or locally and re-upload.
